# results/

Output artifacts generated by running the notebook or training scripts.
Nothing in this directory should be edited by hand — everything here is
reproducible by re-running the pipeline.

## Structure

```
results/
├── figures/    Saved plots (PNG) — pIC50 distribution, correlation heatmaps,
│               benchmark comparison charts, actual-vs-predicted scatter plots,
│               SHAP summary plots, chemical space (PCA/t-SNE) visualizations
├── models/     Serialized trained models — .pkl (sklearn/XGBoost/LightGBM/
│               CatBoost via joblib), .pt (PyTorch state dicts), .zip (TabNet)
└── reports/    Tabular benchmark results — benchmark_results.csv ranking
                all 20+ models by R², RMSE, MAE, and cross-validation scores
```

## Regenerating Results

All files here are produced by `notebooks/QSAR_ML_Models.ipynb` or by running
the CLI benchmark directly:

```bash
make benchmark
```

## Git Tracking Note

Model binaries (`results/models/*.pkl`, `*.pt`, `*.cbm`) are excluded from
version control by `.gitignore` due to their size. Figures and the benchmark
CSV report are small enough to track if you want reproducible reference
outputs committed alongside the code — adjust `.gitignore` if you'd prefer
that behavior.
